Principais Formatações

Negrito/Destaque

Vamos testar a função NEGRITO com a tag STRONG.

Itálico

Vamos testar a função ITÁLICO com a tag EM.

Texto Marcado

Vamos aprender a marcar textos como caneta marca texto com a tag MARK.

Texto Grande e Pequeno

Podemos usar a tag BIG para deixar uma parte do texto com um tamanho maior que o restante (esta tag está descontinuada, logo, o visual studio code não reconhece, porém, funciona)

E também podemos usar a tag SMALL para deixar uma parte do texto com o tamanho um pouco menor do que o restante (esta tag não foi descontinuada como a BIG)

Texto Riscado

Para riscar um texto, usamos a tag DEL

Texto Sublinhado

Para sublinhar um texto, usamos a tag U ou INS.

Texto Sobrescrito

Para deixar um texto ou parte de um texto sobrescrito, como usado em matemática, X^3, usamos a tag SUP, ex: X2 + 3.

Texto Subscrito

Para deixar um texto ou parte de um texto subscrito, como usado em química, H2O, usamos a tag SUB, ex: H2O.
